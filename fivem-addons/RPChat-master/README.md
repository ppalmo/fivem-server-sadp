# RPChat
A Standalone Proximity Roleplaying Chat Resource. 
# Features
- Proximity Chat
- /ooc
- /me (Proximity Chat)
- /ad
- /roll (Proximity Chat)
- /twt
# Credits
- ESX for the Base.
# Installation: 
- Extract the folder to your resources. 
- Rename the folder to "rpchat".
- Add the resource to the startup configuration.
